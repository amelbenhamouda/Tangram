Description détaillée

Espace de nommage geometricShape regroupant les différentes classes de figures utilisées.

Le namespace geometricShape permet de regouper les différentes figures et fonctions extern des class utiliser pour le tangram. Les class du namespace sont shape, rigth_triangle, paralleogram, square.
